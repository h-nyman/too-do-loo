# Too-do-loo

This is a to-do app.

## Quick start
Open index.html in your web browser.

## Features
* Add to-dos
* Complete to-dos
* Remove completed to-dos
* Save to-dos to local storage

## Development guide
Modify HTML in index.html file. 
When you need interaction, add it to main.js. 
Styling should be maintained in main.css.
